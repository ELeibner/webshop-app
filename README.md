## Webshop Application - Full stack (MERN)

Responsive Webshop (e-commerce) full-stack application in React.js, TypeScript, Node.js, Express.js, Redux, Async Thunk, Axios, Jest, MongoDB, Mongoose ODM, MaterialUI
